How I Created This Website

  • Step 1: I used Cloudflare to host my website.
  • Step 2: In Cloudflare, click on Compute (Workers) in the left navigation and go to Workers & Pages.
  • Step 3: Click on Create.
  • Step 4: Select Import a Repository to start by importing an existing Git repository.
  • Step 5: Set up your application. It will ask for details like your Git account and project name. Provide your repository information.
  • Step 6: Finally, click Create and Deploy.
  • Step 7: You can now access your website using the provided domain.
